# Triage labels

The canonical triage labels used by skills in this repo are:

| Role            | Label             |
|-----------------|-------------------|
| Needs triage    | `needs-triage`    |
| Needs info      | `needs-info`      |
| Ready for agent | `ready-for-agent` |
| Ready for human | `ready-for-human` |
| Won't fix       | `wontfix`         |

These are the default strings. If you change them in GitHub, update this file.
